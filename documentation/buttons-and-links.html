Bootstrap (4.2.1) color, size, border radius, and disabled classes can be applied to buttons. For accessibily, Bootstrap's out-of-the-box :hover and :focus styles (box shadow) have been removed in favor of native browser default focus (blue halo).

Additionally, all buttons :hover and :focus states underline text to ensure a noticable state-change when navigating or tabbing through content. This customization is found in scss/_d-enhancements.scss.

Example of the aformentioned accessibility modifications made to buttons with Bootstrap's btn class:

Buttons and links with a disabled class include the css rule cursor: not-allowed;, which on hover, indicates that the button is disabled.

<button class="btn btn-lg btn-block btn-primary disabled" type="button" aria-disabled="true" tabindex="-1">Primary</button>

<a class="disabled" title="Link" aria-disabled="true" tabindex="-1">Link</a>

Icon buttons

Icon buttons are created using the free version of Font Awesome. Add a no-btn-style class to <button>. When using icons without concurrent text, add a title attribute to describe the icon.

Instances of Font Awesome icons should include a role="img" attribute. 

<button class="no-btn-style" type="button">
    <span class="fas fa-search" title="Search" role="img"></span>
</button>

Icons inside buttons and links

To use a Font Awesome icon with any <button> or <a> element, simply add a has-icon class to the parent element. Wrap the button text with <span class="title mx-1"> with two classes: title and mx-1.

  • Link

The title class ensures only the text will be underlined on hover and focus, while the mx-1 class provides even spacing around the text when a right-to-left language is being used.

<button class="btn btn-lg btn-primary has-icon" type="button">
    <span class="fab fa-accessible-icon"role="img"></span>
    <span class="title mx-1" role="img">Accessible</span>
</button>

<a class="has-icon" href="#1" class="text-primary" title="Link">
    <span class="fab fa-accessible-icon font-weight-normal" role="img"></span>
    <span class="title mx-1" role="img">Accessible</span>
</a>

Blended links

Add a blended-link class to an <a> tag to automatically inherit the parent element's color while adding text-decoration:underline; to the link.

Links within articles

For accessibility, all links without a class (unstyled) that are present within #primary-content have text-decoration:underline; applied. This customization is found in scss/_e-supplementary-classes.scss.
